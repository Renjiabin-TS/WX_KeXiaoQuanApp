# 小程序调试指南

## 问题排查清单

### 1. 小程序无法正常显示

#### 可能原因：
- 云开发环境未正确配置
- 环境ID错误
- 云开发服务未开通
- 基础库版本过低

#### 解决步骤：

**步骤1：检查云开发环境**
1. 点击微信开发者工具顶部的「云开发」按钮
2. 如果提示开通，按提示开通云开发服务
3. 在控制台顶部复制环境ID（类似：cloud1-xxxxx）

**步骤2：配置环境ID**
1. 打开 `miniprogram/app.js` 文件
2. 找到第9行的 `env` 配置
3. 将环境ID修改为实际的值：
   ```javascript
   wx.cloud.init({
     env: "你的环境ID",  // 替换为实际的环境ID
     traceUser: true,
   });
   ```

**步骤3：检查基础库版本**
1. 点击微信开发者工具右上角的「详情」
2. 点击「本地设置」标签
3. 确保「调试基础库」版本 ≥ 2.2.3

**步骤4：重新编译**
1. 点击「编译」按钮
2. 或按快捷键 `Ctrl + B`（Windows）或 `Cmd + B`（Mac）

### 2. 测试页面验证

我已经创建了一个测试页面来验证小程序是否能正常运行：

**访问测试页面：**
1. 在微信开发者工具中，点击顶部工具栏的「编译」旁边的选择器
2. 选择「添加编译模式」
3. 页面路径填写：`pages/test/index`
4. 点击「确定」
5. 切换到这个编译模式并编译

**测试页面显示内容：**
- ✅ 成功图标
- "小程序正常运行" 提示
- 当前页面信息

### 3. 查看控制台错误

**步骤：**
1. 在微信开发者工具底部找到「调试器」标签
2. 点击「Console」（控制台）
3. 查看是否有红色错误信息

**常见错误及解决：**

| 错误信息 | 原因 | 解决方案 |
|---------|------|----------|
| "env is not defined" | 环境ID未配置 | 在 app.js 中配置环境ID |
| "云开发环境不存在" | 环境ID错误 | 检查并修正环境ID |
| "请使用 2.2.3 或以上的基础库" | 基础库版本过低 | 升级基础库版本 |
| "集合不存在" | 数据库未创建 | 在云开发控制台创建 users 和 lessons 集合 |

### 4. 降级模式运行

如果云开发暂时无法使用，小程序会自动进入降级模式：

**降级模式特点：**
- 页面能正常显示
- 统计数据显示为 0
- 课时列表显示为空
- 用户可以点击「添加」按钮（但无法真正保存）

**恢复完整功能：**
1. 正确配置云开发环境
2. 重新编译小程序
3. 登录并添加真实数据

### 5. 逐页测试

**测试顺序：**
1. 测试页面（pages/test/index）- 验证基础功能
2. 个人中心（pages/profile/index）- 测试登录功能
3. 统计首页（pages/index/index）- 测试数据展示
4. 日历页面（pages/calendar/index）- 测试日历功能
5. 添加页面（pages/add/index）- 测试添加功能
6. 编辑页面（pages/edit/index）- 测试编辑功能

### 6. 检查网络连接

**确保：**
- 微信开发者工具能访问网络
- 云开发服务网络正常
- 没有防火墙或代理阻止连接

### 7. 清除缓存

如果遇到奇怪的问题，尝试清除缓存：

1. 点击「工具」→「清除缓存」
2. 选择「清除文件缓存」和「清除数据缓存」
3. 重新编译

### 8. 重启开发工具

如果以上方法都不行：

1. 关闭微信开发者工具
2. 重新打开
3. 重新导入项目

## 快速诊断命令

在控制台输入以下命令进行诊断：

```javascript
// 检查云开发是否可用
console.log('云开发可用:', !!wx.cloud);

// 检查基础库版本
console.log('基础库版本:', wx.getSystemInfoSync().SDKVersion);

// 检查环境配置
console.log('全局数据:', getApp().globalData);
```

## 联系支持

如果以上方法都无法解决问题：

1. 截图错误信息
2. 记录操作步骤
3. 查看控制台日志
4. 联系技术支持

## 预期效果

**正常情况下，小程序应该显示：**
- 底部有3个导航标签：「统计」「日历」「我的」
- 统计页面显示课时统计卡片
- 日历页面显示当前月份日历
- 个人中心显示登录界面或用户信息
- 页面样式统一，交互流畅

**降级模式下：**
- 页面能正常显示
- 数据为空或为0
- 不影响页面跳转和UI展示
