# TabBar 图标说明

## 需要准备的图标文件

请在 `images/tabbar` 目录下放置以下图标文件:

### 首页图标
- `home.png` - 未选中状态图标 (81x81px, 建议 40KB 以内)
- `home-active.png` - 选中状态图标 (81x81px, 建议 40KB 以内)

### 日历图标
- `calendar.png` - 未选中状态图标 (81x81px, 建议 40KB 以内)
- `calendar-active.png` - 选中状态图标 (81x81px, 建议 40KB 以内)

### 我的图标
- `profile.png` - 未选中状态图标 (81x81px, 建议 40KB 以内)
- `profile-active.png` - 选中状态图标 (81x81px, 建议 40KB 以内)

## 图标要求
- 图片格式: PNG
- 图片大小: 81x81 像素 (推荐)
- 文件大小: 建议 40KB 以内
- 背景色: 建议透明背景
- 颜色: 未选中状态使用灰色 (#666666), 选中状态使用绿色 (#07C160)

## 临时方案
如果暂时没有图标,可以使用 emoji 作为临时图标:
- 首页: 📊
- 日历: 📅
- 我的: 👤
